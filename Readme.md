# 🎌 MAL XML Viewer

A modern and lightweight **MyAnimeList (MAL) XML Viewer** built with **React, TypeScript, Vite, and Tailwind CSS**. Easily upload your exported MAL XML file to browse your anime collection with a clean, responsive interface featuring search, filters, collection statistics, and detailed anime information.

## ✨ Features

- 📂 Upload MyAnimeList XML files
- 🔍 Fast search and filtering
- 📊 Collection statistics dashboard
- 🖼️ Card & 📋 Table view
- 📖 Detailed anime information
- 📜 Raw XML viewer
- 📤 Export parsed data as JSON
- 📱 Responsive design

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/MAL-XML-Viewer.git
cd MAL-XML-Viewer
npm install
npm run dev
```
